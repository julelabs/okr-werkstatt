Was diese Website verarbeitet

Die OKR-Werkstatt ist eine statische Website. Sie verwendet kein Analyse- oder Werbetracking und setzt keine Cookies.

Der interaktive Lernpfad speichert den gewählten Lernstand ausschließlich lokal im Browser (localStorage). Diese Angaben werden nicht an Julia Breyer übertragen. Sie können durch das Löschen der Website-Daten im Browser oder über die Funktion „Pfad wechseln“ entfernt werden.

Hosting und Auslieferung der Website

Die technischen Dateien dieser Website werden über GitHub Pages bereitgestellt, einen Dienst von GitHub. Der Website ist Cloudflare vorgeschaltet. Cloudflare übernimmt unter anderem die Domainauflösung sowie die sichere und beschleunigte Auslieferung der Inhalte.

Beim Aufruf der Website verarbeiten GitHub und Cloudflare technisch notwendige Verbindungs- und Verkehrsdaten. Dazu können insbesondere IP-Adresse, aufgerufene Adresse, Datum und Uhrzeit des Zugriffs, Geräte- und Browserinformationen, Systemkonfiguration sowie Informationen über den Datenverkehr gehören.

Für die dortige Verarbeitung personenbezogener Daten ist der jeweilige Anbieter verantwortlich. Bitte gib in den Bot keine vertraulichen oder personenbezogenen Informationen ein.
